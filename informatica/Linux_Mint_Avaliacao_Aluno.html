1. O QUE É O LINUX?

O Linux é um sistema operacional de código aberto criado em 1991 pelo estudante finlandês Linus Torvalds. Diferente do Windows e macOS, pode ser copiado, modificado e distribuído livremente. É a base de servidores web, smartphones Android, smart TVs e supercomputadores.

🌍 Mais de 96% dos servidores da internet rodam Linux. Google, NASA, Netflix e Instagram funcionam em servidores Linux.
  • Kernel: núcleo que faz a comunicação entre hardware e programas.
  • Distribuição (distro): sistema completo: kernel + programas + interface gráfica. Ex.: Ubuntu, Debian, Linux Mint.
  • Ambiente Gráfico: interface visual. O Mint usa o Cinnamon por padrão.
2. SISTEMA DE ARQUIVOS

No Linux tudo começa na raiz /. Não há C:\, D:\ como no Windows — há uma única árvore de diretórios.

DiretórioFunção
/Raiz — ponto de partida de toda a estrutura.
/homePasta pessoal de cada usuário.
/etcArquivos de configuração do sistema.
/bin, /usr/binProgramas acessíveis a todos os usuários.
/varLogs do sistema e dados variáveis.
/tmpArquivos temporários — limpos a cada reinicialização.
/mediaPonto de montagem de pen drives, CDs etc.
/devRepresentação dos dispositivos como arquivos.
3. O TERMINAL

Abra com Ctrl+Alt+T. Estrutura de um comando:

Terminal
# comando [opções] [argumentos]
$ ls -la /home
ComandoO que faz
pwdMostra o diretório atual.
ls -laLista arquivos com detalhes e ocultos.
cd pastaEntra em uma pasta. cd .. volta. cd ~ vai para /home.
mkdir nomeCria uma pasta.
touch arqCria um arquivo vazio.
cp orig destCopia arquivo (original mantido).
mv orig destMove/renomeia (original some).
rm arqRemove arquivo. rm -r remove pasta.
cat arqExibe conteúdo no terminal.
ip aExibe endereço IP e interfaces de rede.
ping hostTesta conectividade.
ps auxLista todos os processos.
top / htopMonitor de processos em tempo real.
kill PIDEncerra processo pelo número PID.
df -hEspaço em disco.
free -hUso de memória RAM.
4. PERMISSÕES
Os 3 tipos de permissão
🔵 r (read) — Leitura: pode abrir e ver o conteúdo.
🟡 w (write) — Escrita: pode modificar ou apagar.
🟢 x (execute) — Execução: pode rodar como programa.
⚫ – (hífen) — Permissão não concedida.
ls -l
-rwxr-xr-- 1 marcos alunos 4096 Jun 01 script.sh
[tipo][dono=rwx][grupo=r-x][outros=r--]
ValorPermissãoSignificado
7rwxLeitura + Escrita + Execução
6rw–Leitura + Escrita
5r-xLeitura + Execução
4r––Somente Leitura
0–––Sem permissão
chmod
$ chmod 755 script.sh # dono:rwx grupo:r-x outros:r-x
$ chmod 644 arquivo.txt # dono:rw- grupo:r-- outros:r--
5. SHELL SCRIPT

Arquivo com sequência de comandos executados de uma vez.

script.sh
#!/bin/bash
# Comentário
echo 'Olá, Mundo!'
NOME='Marcos'
echo "Bem-vindo, $NOME!"
if [ -f /etc/os-release ]; then echo 'Existe!'; fi
executar
$ nano script.sh
$ chmod +x script.sh
$ ./script.sh
6. GERENCIAMENTO DE PACOTES (APT)
Comandos APT principais
sudo apt update → Atualiza lista de pacotes disponíveis.
sudo apt upgrade → Instala atualizações dos programas.
sudo apt install nome → Instala um programa.
sudo apt remove nome → Remove um programa.
sudo apt autoremove → Remove pacotes desnecessários.
