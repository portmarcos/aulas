1. O que é o Linux?
Conceitos fundamentais e distribuições

O Linux é um sistema operacional de código aberto criado em 1991 pelo estudante finlandês Linus Torvalds. Diferente do Windows e macOS, pode ser copiado, modificado e distribuído livremente. É a base de servidores web, smartphones Android e supercomputadores.

Mais de 96% dos servidores da internet rodam Linux. Google, NASA, Netflix e Instagram funcionam em servidores Linux.
  • Kernel: núcleo que faz a comunicação entre hardware e programas.
  • Distribuição (distro): sistema completo — kernel + programas + interface gráfica. Ex.: Ubuntu, Debian, Linux Mint.
  • Ambiente Gráfico: interface visual com janelas e menus. O Mint usa o Cinnamon por padrão.
2. Sistema de Arquivos
Estrutura de diretórios do Linux

No Linux tudo começa na raiz /. Não há C:\, D:\ como no Windows — há uma única árvore de diretórios.

DiretórioFunção
/Raiz — ponto de partida de toda a estrutura.
/homePasta pessoal de cada usuário.
/etcArquivos de configuração do sistema.
/bin, /usr/binProgramas acessíveis a todos os usuários.
/varLogs do sistema e dados variáveis.
/tmpArquivos temporários — limpos a cada reinicialização.
/mediaPonto de montagem de pen drives, CDs.
3. O Terminal
Abra com Ctrl + Alt + T
bash — Linux Mint
# estrutura: comando [opções] [argumentos]
$ ls -la /home
$ pwd
$ cd /etc
$ mkdir projetos
$ chmod 755 script.sh
ComandoO que faz
pwdMostra o diretório atual.
ls -laLista arquivos com detalhes e ocultos.
cd pastaEntra em uma pasta. cd .. volta. cd ~ vai para /home.
mkdir nomeCria uma pasta.
cp orig destCopia arquivo (original mantido).
mv orig destMove ou renomeia (original some).
rm arquivoRemove arquivo. rm -r remove pasta inteira.
kill PIDEncerra processo pelo número PID.
sudo apt installInstala um programa.

5. Shell Script
Automatizando tarefas no terminal
relatorio.sh
#!/bin/bash
# Comentário — não é executado
echo 'Olá, Mundo!'
NOME='Marcos'
echo "Bem-vindo, $NOME!"
date
free -h
df -h /
Para executar: chmod +x script.sh → ./script.sh
