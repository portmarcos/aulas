O IP é um código numérico que identifica cada dispositivo em uma rede — como um "CEP" digital.

IPv4 — 32 bits
Formato: XXX.XXX.XXX.XXX — 4 grupos decimais (0-255)
Exemplo: 192.168.1.10
Total de endereços: ~4,3 bilhões
IPv6 — 128 bits
Formato: XXXX:XXXX:XXXX:XXXX:XXXX:XXXX:XXXX:XXXX — 8 grupos hexadecimais
Exemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
Total: ~340 undecilhões (3,4 × 10³⁸)

DNS traduz nomes de domínio em IPs — a "lista telefônica" da internet.

Você digita: www.google.com
DNS resolve: 142.250.218.14 → navegador acessa o IP
  • DNS público: 8.8.8.8 (Google), 1.1.1.1 (Cloudflare)
  • DNS Forwarder: DNS interno repassa consultas externas para DNS externo
  • Bloqueio por DNS: servidor configurado para não resolver certos domínios
⚠ Sem DNS: sites abrem pelo IP mas não pelo nome de domínio.

TCP
  • Orientado a conexão (handshake)
  • Garante entrega e ordem
  • Mais lento
  • Uso: HTTP, e-mail, banco online
UDP
  • Sem conexão prévia
  • Sem garantia de entrega
  • Muito rápido
  • Uso: streaming, VoIP, jogos

Sockets: o TCP identifica cada conexão por IP de origem + porta de origem + IP de destino + porta de destino. Cada cliente usa uma porta de origem diferente (ephemeral port), permitindo múltiplas conexões simultâneas ao mesmo servidor.

💡 Dados críticos (banco, e-mail) → TCP. Dados em tempo real (streaming, jogos) → UDP.
🎯 Questões: 13 · 16 · 21 · 28

NAT permite que múltiplos dispositivos com IPs privados compartilhem um único IP público.

1. PC (192.168.1.10:52000) envia pacote para google.com
2. Roteador substitui pelo IP público (200.10.1.1:45001)
3. Google responde para 200.10.1.1:45001
4. Roteador consulta tabela → encaminha para 192.168.1.10:52000
🏢 Analogia: edifício com 40 funcionários usando o mesmo ramal externo, cada um com ramal interno diferente.

A máscara define qual parte do IP identifica a rede e qual identifica o dispositivo (host).

Com /24, os 3 primeiros octetos identificam a rede:
192.168.1.10 e 192.168.1.50 → mesma rede ✅ comunicam diretamente
192.168.1.10 e 192.168.2.10 → redes diferentes ❌ precisam do gateway
